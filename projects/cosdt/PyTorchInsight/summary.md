> PyTorch 社区周期性洞察分析系统，自动采集 PyTorch 仓库的 Issue/PR/讨论数据，分析社区活跃度、贡献者趋势和热点话题，为团队制定上游贡献策略提供数据支撑。

## 项目介绍
> PyTorch 社区数据洞察工具。核心场景：(1) 周期性采集 PyTorch GitHub 数据 (2) 分析贡献者活跃度和趋势 (3) 生成社区健康报告和热点话题识别。
